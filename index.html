2048(Adapted version)

This is an adapted version of 2048, the game created by Gabriele Cirulli. The rules are pretty much the same, but some changes have been added. Now the multi-collapse feature is enabled, that is, if there is a 8-4-2-2 in some row, then first the pair of twos will collapse, creating a 4, then the pair of fours will collapse, and finally the pair of eights - all simultaneously.
